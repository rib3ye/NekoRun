How it works

Three steps. Two of them are setup.

01

Tell it where

Click the cat → Set Upload Destination… and give it you@yourserver plus a remote path. It remembers.

02

Maybe add a hook

Optionally set a shell command that runs after every upload — chmod things, restart things, notify things.

03

Drop stuff on the cat

Files, folders, whatever. The cat wiggles while scp -r does its thing. That wiggle is the progress bar.
